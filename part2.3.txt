                                                    <h3>Description</h3>
                            <div id="ord_general">
                                <div class="row creator-row" id="creator-row-e5d90" data-type="full">
        <div class="col-md-12">
        <p>The dataset contains <span>LEGO bricks sets item count and pricing history for AI-based set pricing prediction</span><span>.</span></p>
<p><span>The data was downloaded from three sources. LEGO sets retail prices, release dates, and IDs were downloaded from Brickset.com, where one can find the ID number of each set that was released by Lego and its retail prices. The current status of the sets was downloaded from Lego.com and the retail prices for Poland and prices from aftermarket transactions were downloaded from promoklocki.pl. The data was merged based on the official LEGO set ID.</span></p>
<p><span>The data is composed of one aggregated table stored in an XLSX file named lego_final_data.xlsx. All data was scrapped from lego.com, brickset.com, and promoklocki.pl websites. The table contains the following columns:</span><span> </span></p>
<ul>
<li><span>setID – internal Brickset.com LEGO set identification number,</span><span> </span></li>
<li><span>number – official LEGO set ID,</span><span> </span></li>
<li><span>numberVariant – official LEGO set sub variant (e.g. different minifigure hidden in a random bag),</span></li>
<li><span>name – official LEGO set name,</span></li>
<li><span>year – the set release year,</span></li>
<li><span>theme – official name of the set theme,</span></li>
<li><span>themeGroup – official name of the set themes grup (if available),</span></li>
<li><span>subtheme – official name of the set sub-theme (if available),</span></li>
<li><span>category – brickset.com internal set type,</span></li>
<li><span>released – indicates whether the set was officially released (1) or not (0),</span></li>
<li><span>pieces – number of parts in the set,</span></li>
<li><span>minifigs – number of minifigures in the set,</span></li>
<li><span>ownedBy – number of brickset.com users claiming that he or she owns the set,</span></li>
<li><span>wantedBy – number of brickset.com users claiming that he or she wants to buy the set,</span></li>
<li><span>rating – average set rating according to brickset.com users,</span></li>
<li><span>reviewCount – number of the set reviews written by brickset.com users,</span></li>
<li><span>packagingType – type of packaging for the set (if specified),</span></li>
<li><span>availability – indicates whether the set was available in retail shops or only on official LEGO shop web site,</span></li>
<li><span>instructionsCount – number of books with building instructions added to the set,</span></li>
<li><span>minAge – LEGO recommended minimal user age for the set,</span></li>
<li><span>maxAge – LEGO recommended maximal user age for the set (either not specified or 99),</span></li>
<li><span>tags – list of brickset.com assigned set tags,</span></li>
<li><span>LastUpdated – the date and time of the last update of the data in brickset.com in ISO 8601 format,</span></li>
<li><span>urlRetailPriceCheckPLN – URL where retail price in PLN was downloaded from,</span></li>
<li><span>US_retailPrice – retail price in United States in US dollars,</span></li>
<li><span>US_dateFirstAvailable – date and time when the set became available in United States in ISO 8601 format,</span></li>
<li><span>US_dateLastAvailable – date and time when the set stopped being officially available in United States in ISO 8601 format,</span></li>
<li><span>UK_retailPrice – retail price in United Kingdom in GBP,</span></li>
<li><span>UK_dateFirstAvailable – date and time when the set became available in United Kingdom in ISO 8601 format,</span></li>
<li><span>UK_dateLastAvailable – date and time when the set became available in United Kingdom in ISO 8601 format,</span></li>
<li><span>CA_retailPrice – retail price in Canada in Canadian dollars,</span></li>
<li><span>CA_dateFirstAvailable – date and time when the set became available in Canada in ISO 8601 format,</span></li>
<li><span>CA_dateLastAvailable – date and time when the set became available in Canada in ISO 8601 format,</span></li>
<li><span>DE_retailPrice – retail price in Germany in EUR,</span></li>
<li><span>DE_dateFirstAvailable – date and time when the set became available in Germany in ISO 8601 format,</span></li>
<li><span>DE_dateLastAvailable – date and time when the set became available in Germany in ISO 8601 format,</span></li>
<li><span>PL_retailPrice – retail price in Poland in PLN,</span></li>
<li><span>Date – year and month for which the PriceMonthPLN is given,</span></li>
<li><span>priceMonthPLN – price in PLN read from promoklocki.pl for year and month specified in Date column,</span></li>
<li><span>status – official status of the set (if available) in LEGO web shop,</span></li>
<li><span>urlRetailPriceHistoryPLN – URL containing retail and aftermarket price changes from the day of the release of the set, in PLN.</span><span> </span></li>
</ul>
